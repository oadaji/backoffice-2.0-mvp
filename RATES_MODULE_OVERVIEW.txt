EXECUTIVE SUMMARY
================================================================================
The Rates module is a freight rate management system that allows the OnePort 365
team to view, search, filter, and compare freight rates from multiple carriers
across different shipping methods. This module serves as a centralized rate
database for the operations team to quickly find competitive rates and provide
quotes to customers.


WHAT THE RATES MODULE DOES
================================================================================

PRIMARY FUNCTIONS:
1. Display freight rates from multiple carriers in a searchable table format
2. Filter rates by carrier, rate type, route, and expiry status
3. Show detailed rate breakdowns including all surcharges and fees
4. Track rate expiry dates and highlight expiring/expired rates
5. Support multiple freight types: Ocean, Air, Export Haulage, Import Haulage,
   and Other Charges

USER WORKFLOWS:
- Operations team searches for rates based on customer requirements
- Compare rates from different carriers side-by-side
- View detailed charge breakdowns before providing customer quotes
- Track which rates are about to expire
- Export rate data to CSV for reporting/analysis


FREIGHT TYPES SUPPORTED
================================================================================

1. OCEAN FREIGHT (15 rates currently)
   - Port-to-port container shipping
   - Container types: 20GP, 40GP, 40HC, 40RF (refrigerated)
   - Shows rates per container type
   - Includes major carriers: MSC, Maersk, CMA CGM, Hapag-Lloyd, COSCO,
     Evergreen, ONE, PIL, ZIM

2. AIR FREIGHT (4 rates currently)
   - Airport-to-airport cargo shipping
   - Cargo types: General Cargo, Perishables
   - Shows rates per kilogram ($/kg) in weight brackets: +45kg, +100kg, +300kg
   - Includes carriers: Emirates SkyCargo, Turkish Cargo, Qatar Airways Cargo,
     Air France Cargo

3. EXPORT HAULAGE (2 rates currently)
   - Inland trucking from customer location to port/airport
   - Nigeria-based domestic transport to export terminals
   - Rates in Nigerian Naira (₦)
   - Equipment: 20GP, 40HC containers

4. IMPORT HAULAGE (1 rate currently)
   - Port/airport to customer final destination
   - Domestic delivery from import terminals
   - Rates in Nigerian Naira (₦)

5. OTHER CHARGES (1 rate currently)
   - Terminal handling charges (THC)
   - Port fees and surcharges
   - Documentation fees
   - Gate fees, scanner fees, lifting charges


DATA MODEL - WHAT'S IN A RATE
================================================================================

Each rate record contains:

BASIC INFORMATION:
- id: Unique identifier (r1, r2, r3...)
- type: Freight category (ocean, air, haulage-export, haulage-import, other)
- carrier: Shipping line/airline/trucking company name

ROUTE INFORMATION:
- pol: Port/Airport of Loading (UN/LOCODE format, e.g., NGAPP, CNSHA)
- origin: Human-readable origin location (e.g., "Nigeria", "China")
- pod: Port/Airport of Discharge (UN/LOCODE format)
- dest: Human-readable destination location

PRICING INFORMATION:
- rate20: Rate for 20-foot container OR +45kg air freight
- rate40: Rate for 40-foot container OR +100kg air freight
- rate40hc: Rate for 40-foot high-cube container OR +300kg air freight
- Currency: USD for ocean/air, NGN (₦) for haulage/other charges

OPERATIONAL DETAILS:
- equip: Equipment/cargo type (20GP, 40HC, General Cargo, Perishables, etc.)
- rateType: Pricing method (spot, contract, all in, published)
- transit: Transit time in days (0.5 for local haulage, 1 for air, 14-35 for ocean)
- expiry: Rate validity end date (YYYY-MM-DD format)

CHARGE BREAKDOWN:
- charges: Array of individual cost components
  * Ocean freight base rate (OFR)
  * Bunker Adjustment Factor (BAF) - fuel costs
  * Peak Season Surcharge (PSS)
  * Emergency Bunker Surcharge (EBS)
  * Terminal Handling Charges - Origin (THC-O)
  * Terminal Handling Charges - Destination (THC-D)
  * Documentation fees (DOC)
  * Security charges
  * Equipment-specific charges (reefer, cool chain, etc.)


RATE TYPES EXPLAINED
================================================================================

1. SPOT RATE
   - Short-term market rate based on current supply/demand
   - Typically valid for 7-30 days
   - Subject to rapid changes
   - Used for one-off shipments

2. CONTRACT RATE
   - Long-term negotiated rate with carrier
   - Valid for 3-12 months
   - Volume commitment required
   - More stable pricing

3. ALL IN RATE
   - Includes all charges bundled together
   - Origin + freight + destination charges included
   - No hidden fees
   - Easier for customer quotes

4. PUBLISHED RATE
   - Carrier's standard tariff rate
   - Publicly available pricing
   - Starting point for negotiations

TECHNICAL IMPLEMENTATION DETAILS
================================================================================

TECHNOLOGY STACK:
- Vanilla JavaScript (no framework)
- Single HTML file (index.html)
- Inline CSS with custom properties for theming
- Local data storage (rates array in JavaScript)
- No backend API (all data is client-side)

STATE MANAGEMENT:
- activeSection: 'rates' to show this module
- rateType: Currently selected tab (ocean/air/haulage-export/etc.)
- rateSearch: Current search query
- rateCarrier: Selected carrier filter
- rateRateType: Selected rate type filter

URL PARAMETERS (NEW):
- ?section=rates - Load directly to rates section
- ?rateType=air - Pre-select specific freight type
- Enables deep linking and Figma captures

CARRIER BRANDING:
- SVG logos stored in /assets/carriers/
- Supported carriers: MSC, Maersk, CMA, Hapag, COSCO, Evergreen, ONE, PIL, ZIM
- Carrier-specific color coding in UI
- Fallback to carrier initials if logo unavailable

KEY BUSINESS RULES
================================================================================

RATE VALIDITY:
- Expired rates are not hidden but are visually marked
- Rates expiring within 14 days show warning indicator
- Operations team responsible for requesting rate renewals
- Always check expiry before providing customer quote

PRICING TRANSPARENCY:
- All rates should show complete charge breakdown
- No hidden fees
- Clear distinction between base rate and surcharges
- Total should match sum of all charge components

CARRIER RELATIONSHIPS:
- Contract rates require minimum volume commitment
- Spot rates available to all customers
- Preferred carrier partnerships may offer better rates
- Some carriers offer equipment type restrictions

ROUTE COVERAGE:
- Focus on Nigeria import/export lanes
- Major trading partners: China, UAE, Europe, USA
- Regional coverage for West Africa
- Air freight for time-sensitive cargo

KNOWN LIMITATIONS & CONSTRAINTS
================================================================================

CURRENT PROTOTYPE LIMITATIONS:
- No real carrier rate feeds
- Manual data entry required
- No rate change history
- No user permissions/access control
- Cannot handle rate amendments or corrections
- No support for special agreements or contracts
- No validation of port codes or carrier names
- Limited to 5 freight types
- No support for FCL/LCL distinction
- No dangerous goods or special cargo handling
- No multi-leg routing (transshipment ports)

SCALABILITY CONCERNS:
- Client-side filtering won't scale beyond ~1,000 rates
- No pagination or lazy loading
- Large rate tables may cause browser performance issues
- Export function may timeout with large datasets
